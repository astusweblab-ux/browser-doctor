# Privacy Policy

Browser Doctor does not collect, transmit, sell, or store personal data on external servers.

## What the extension accesses

- Open tabs metadata (title, URL, tab state) to compute diagnostics
- Tab favicon URLs provided by Chrome (`tab.favIconUrl`) for UI display
- Installed extensions list (name/version) for visibility in popup
- Local extension storage for diagnostics history and notification cooldowns
- Notification API to show local system alerts about high browser load

## Data storage

All data is stored locally in `chrome.storage.local` and remains on the user's device.
No remote analytics, no external backend, no tracking scripts, no remote code.

## Data sharing

No user data is shared with third parties by Browser Doctor.
